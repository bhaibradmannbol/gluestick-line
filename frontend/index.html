Project Scope

The line models glue core insertion, twist mechanism assembly, outer body mounting, and cap placement. It tracks machine state, product status, timestamps, defect reasons, and production output through a clean operator-facing HMI.

Project Architecture

Frontend HMI

A single-page glassmorphism dashboard that serves as both a working HMI and a public showcase website.

FastAPI Backend

Backend routes provide machine state, product status, and operator actions including start, stop, and reset.

Simulation Engine

A Python production model simulates four assembly stages and injects realistic defect conditions.

InfluxDB Telemetry

Time-series measurements capture finished products and machine state changes for analytics.

Grafana Analytics

Grafana displays live metrics such as output totals, defect counts, and trend-based quality insights.

Built for a university Advanced Programming Project with a deployable frontend, local backend workflow, and future expansion path.
