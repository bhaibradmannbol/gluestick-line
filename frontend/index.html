Glue Stick Production Line Dashboard

A live industrial monitoring interface for a simulated four-stage glue stick assembly line. This dashboard combines a Python backend, real-time HMI controls, InfluxDB logging, and Grafana-ready production telemetry in one clear project view.

Project Scope

Product flow includes glue core insertion, twist mechanism assembly, outer body mounting, and cap placement. The system tracks machine state, product status, timestamps, defect reason, and production totals for demo-ready quality monitoring.

Project Architecture

Frontend HMI

Single-page dashboard for live controls, machine visibility, defect reporting, and project presentation.

FastAPI

Backend routes expose status plus operator actions for start, stop, and reset.

Simulation

Four assembly stages model glue stick production flow, faults, and product lifecycle state.

InfluxDB

Time-series logging stores production results and machine-state events for analysis.

Grafana

Dashboard layer visualizes totals, defects, and live production quality metrics.

Built for a university Advanced Programming Project with a submission-friendly architecture and public-demo direction.
